js里的Map

本文最后更新于：17 天前

本文介绍 JavaScript 中的 Map

介绍

Map 是 js 原生支持的对象，用于保存键值对，并且是有序的。
Map 对象在迭代时按插入顺序返回一个 [key, value] 数组。
任何值都可以作为 Map 的键或值，其中键要求 唯一 。键通过运算符 === 比较是否相同，其中特殊的是虽然 NaN !== NaN ，但是 NaN 也不可以重复。

如何使用

实例化

通过 new 实例化一个 Map 对象。

属性

size 属性表示 Map 对象的条目数。

方法

方法名 作用 返回值 备注 例子
clear 移出对象所有键值对
delete(key) 删除key对应的键值对 若存在对应key返回true，否则返回false
forEach(callback) 按照插入顺序为每一对键值对调用一次callback
get(key) 获取键对应值 键存在返回对应值，否则返回undefined
set(key, value) 设置键对应值 返回该Map对象
has(key) 判断对象是否包含对应键 键存在返回true，否则返回false
keys 按照插入顺序获得所有的键 返回一个新的Iterator对象
values 按照插入顺序获得所有的值 返回一个新的Iterator对象

Map 与 Array 的关系

可以通过一个二维键值对数组构造一个 Map 对象：

let kvArray = [["keyone","valueone"],["keytwo","valuetwo"]];
let map = new Map(kvArray);

因为 Map 是一个可迭代对象，可以使用 Array.from 将其转换为 Array 对象：

let toArray = Array.from(map);

也可以使用 展开运算符，简化上面的写法：

let toArray = [...map];

值得注意的是将数组转换为 Map 时，数组中对应键的值重复的项目会被合并，后面的会覆盖前面的，以保证 Map 对象键的唯一。

let testMap = new Map([
    [1,"valueone"],
    [1,"valuetwo"],
    [2,"valuethree"]
]);
// testMap的值为 {1 => "valuetwo", 2 => "valuethree"}
let otherMap = new Map([
    [2,"valuefour"],
    [3,"valuefive"]
]);
let map = new Map([...testMap, ...otherMap]);
// map的值为 {1 => "valuetwo", 2 => "valuefour", 3 => "valuefive"}

注意事项

不建议为 Map 设置对象属性：

let map = new Map();
map["keyone"] = 1;
// 这样做会导致一些错误
