做个刮一刮了解canvas

本文最后更新于：2 个月前

本文将介绍实现一个刮一刮 Demo，帮助了解前端得画布元素。

Demo

布局和样式

将一个刮一刮分为两层，.award 层用来展示奖品内容，.coating 层使用画布元素用来表示涂层。当然你也可以只使用 canvas 展示涂层和奖品，但是我更喜欢分别展示方式。因为将奖品区域独立出来，布局上会更加方便。另外你会发现我将奖品区域设置样式不可见，这是由于网络加载情况和绘制时机可能会导致奖品区域在涂层绘制前先展示出来，你应该不希望这种情况发生。 

<div class="scrape-award">
    <div class="award" style="visibility: hidden;">
        <span>奖品是什么呢</span>
    </div>
    <canvas class="coating"></canvas>
</div>
.scrape-award {
    position: relative;
}
.scrape-award .award {
    height: 150px;
    line-height: 150px;
    text-align: center;
    font-size: 20px;
}
.scrape-award .coating {
    z-index: 2;
    position: absolute;
    left: 0;
    top: 0;
}

逻辑

下面的函数中涉及的全局变量： 

var canvas = null; // canvas对象
var ctx = null; // 画布上下文
var width = 0; // 画布宽度
var height = 0; // 画布高度

更好的做法是将将整个刮一刮封装成一个类，这里为了方便，直接声明为全局变量，并不推荐这种方式。

绘制涂层

window.onload = function () {
    var awardDom = document.getElementsByClassName("award")[0];
    canvas = document.getElementsByClassName("coating")[0];
    ctx = canvas.getContext("2d");
    var rect = awardDom.getBoundingClientRect();
    width = rect.width;
    height = rect.height;
    // 设置画布大小
    canvas.width = width;
    canvas.height = height;
    // 绘制涂层
    scrapeAward.ctx.fillStyle = "#eee";
    scrapeAward.ctx.fillRect(0, 0, scrapeAward.width, scrapeAward.height);
    // 绘制涂层上的文字
    scrapeAward.ctx.fillStyle = "#000";
    scrapeAward.ctx.font="20px Arial";
    scrapeAward.ctx.fillText("测试文本测试文本测试文本测试文", 0, height/2-10, width);
    // 显示奖品层
    awardDom.style.visibility = "";
}

处理 touch 事件

当用户在涂层上滑动时，需要将抹除对应坐标的涂层，所以要在响应 canvas 的 touchmove 事件之前，将绘制新图形时的合成操作类型修改为 destination-out。 

window.onload = function () {
    ...
    ctx.globalCompositeOperation = "destination-out";
    canvas.addEventListener("touchmove", touchMove);
}
function touchMove(e) {
    /* 阻止默认事件 */
    e.preventDefault();
    if (e.changedTouches) {
        let rect = document.getElementsByClassName("award")[0].getBoundingClientRect();
        let pos = e.changedTouches[0];
        let x = pos.clientX - rect.x,
            y = pos.clientY - rect.y;
        scrapeAward.ctx.beginPath();
        scrapeAward.ctx.arc(x, y, 20, 0, Math.PI * 2);
        scrapeAward.ctx.fill();
    }
}

判断已刮涂层百分比

一般我们需要在用户刮出大部分内容后，自动清除剩下的涂层，所以需要判断已刮涂层的百分比。
可以通过 getImageData 获取画布像素矩阵数据，然后判断其中的不透明度等于 0 即为透明，这里取 256 的一半 128 即为半透明。

/* 去除全部涂层 */
function handleFilledPercentage(percentage) {
    percentage = percentage || 0;
    if (parseInt(percentage) > 50) {
        ctx.clearRect(0, 0, width, height);
        /* 或者直接删除画布 */
        canvas.remove();
    }
}
/* 获取已刮得涂层百分比 */
function getFilledPercentage() {
    let imgData = ctx.getImageData(0, 0, width, height);
    let pixels = imgData.data;
    let transPixels = [];
    for (let i = 3; i < pixels.length; i += 4) {
        if (pixels[i] < 128) {
            transPixels.push(pixels[i]);
        }
    }
    return transPixels.length / (pixels.length / 4);
}
