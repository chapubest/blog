jQuery概览

本文最后更新于：1 小时前

本文补充整理 jQuery 的基本内容

介绍

jQuery 是一个快速、小型且功能丰富的 JavaScript 库，通过 轻量的写法实现复杂的功能，包含以下基本功能：

  • DOM 操作
  • CSS 操作
  • 事件函数
  • JavaScript 特效和动画
  • AJAX
  • Utilities
  • 其他扩展插件

简略写法

$ === jQuery;

window.onload 和 $(document).ready 的区别

window.onload $(document).ready
执行时间 网页全部加载完 网页 DOM 加载完
执行次数 1 次 多次
写法 window.onload = function(){} jQuery(document).ready(function($){})
简写 无 $(function(){});

设置 body 的 onload 事件和 window.onload 事件只会执行一个，都是在 body 内容加载结束执行
按照顺序，后面的 onload 会覆盖掉前面的 onload

选择器

XPath 选择器和 CSS 选择器？

DOM 对象可通过 $() 变成 jQuery 对象。常见选择器使用例:

  1. 选择所有 p 标签
    $("p") 单引号也可以
  2. 选择 id 为 test 的标签
    $("#test")
  3. 选择所有 class 为 test 的标签
    $(".test")
  4. 选择所有标签
    $("*")
  5. 选择当前元素
    $("this")
  6. 选择所有 class 为 test 的 p 标签
    $("p.test")
  7. 选择所有 class 为 test 的 p 标签
    $("p.test")
  8. 选择所有 class 同时包含 test1,test2 的元素
    $(".test1.test2")
  9. 选择所有 ul 的元素下的所有 li 元素
    $("ul li")
  10. 选择所有父元素是 div 的 p 标签
    $("div>p")
  11. 选择第一个 ul 的第一个 li 标签
    $("ul li:first")
  12. 选择每个 ul 的第一个 li 标签
    $("ul li:first-child")
  13. 选择属性带有 href 的元素
    $("[href]")
  14. 选择 name 属性为 email 的元素
    $("[name='email']")
  15. 选择 name 属性为 e 开头的元素
    $("[name^='e']")
  16. 选择 name 属性为 e 结尾的元素
    $("[name$='e']")
  17. 选择 name 属性不为 email 的元素
    $("[name!='email']")

DOM 事件

hover() 事件

$("#p1").hover(fun1, fun2) 鼠标移入触发 fun1 ，鼠标移出触发 fun2

效果

hide() 和 show()

hide() 和 show() 等效 display: none 是不占位的
完整用法:
hide(speed, way, callback)
speed 表示执行速度，可选参数：”slow”,”fast”,(ms)
way 表示动画方式，可选参数：”linear”,”swing”
callback 表示完成后执行函数

直接使用 toggle() 自动实现 hide() 和 show()

fade()

参数与 hide() 类似

  • fadeIn(speed, callback)
  • fadeOut(speed, callback)
  • fadeToggle(speed, callback)
  • fadeTo(speed, opacity(0-1), callback) 渐变到指定不透明度

slide()

滑动

  • slideDown(speed, callback)
  • slideUp(speed, callback)
  • slideToggle(speed, callback)

自定义动画 animate()

animate({params}, speed, callback)
params 参数定义目标 CSS 属性
可使用相对值，例如+=， -=
也可使用预定值，例如 “toggle”
使用 Camel 标记法 书写所有的属性名
不包含颜色，transform，需额外下载插件
支持队列，可以编写多个 animate，jQuery 会自动逐一调用
stop() 停止动画效果
stop(stopAll(false), gotoEnd(false))
stopAll 表示清除动画队列
gotoEnd 表示立即完成当前动画

链(Chaining)

将动作和方法链接在一起

$("#p1").css("color", "red").slideUp(2000).slideDown(2000);

  • html() 获取|设置内容

  • attr() 获取|设置属性

回调函数可以获得当前元素下标&原始值

  • append()

  • after()

可添加多个元素

  • remove() 删除该元素及子元素，可传选择器参数

遍历

  1. 祖先
  • parent() 父元素
  • parents() 所有祖先元素
  • parentUntil() 给定元素之间的祖先元素
  1. 子孙

  1. 同胞
  • siblings() 所有同胞
  • next()
  • nextAll()
  • nextUntil()
  • prev()
  • prevAll()
  • prevUntil()
  1. 过滤
  • first()
  • last()
  • eq() 方法返回被选元素中带有指定索引号的元素
  • filter()
  • not()

AJAX
